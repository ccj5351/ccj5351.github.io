C++ Interview Questions

Index Questions Short Answer
1 What is the full form of OOPS? Object Oriented Programming System.
2 What is a class? Class is a blueprint which reflects the entities attributes and actions. Technically defining a class is designing an user defined data type.
3 What is an object? An instance of the class is called as an object.
4 List the types of inheritance supported in C++. Single, Multilevel, Multiple, Hierarchical and Hybrid.
5 List the modes of inheritance 1) Public mode (CCJ: identity map): If we derive a sub class from a public base class. Then the public member of the base class will become public in the derived class and protected members of the base class will become protected in derived class. 2) Protected mode (CCJ: protect children): If we derive a sub class from a Protected base class. Then both public member and protected members of the base class will become protected in derived class. 3) Private mode (CCJ: private 'takes' all): If we derive a sub class from a Private base class. Then both public member and protected members of the base class will become Private in derived class.
6 What is the role of protected access specifier? If a class member is protected then it is accessible in the inherited class. However, outside the class, both the private and protected members are not accessible.
7 What is encapsulation? The process of binding the data and the functions acting on the data together in an entity (class) called as encapsulation.
8 What is abstraction? Abstraction refers to hiding the internal implementation and exhibiting only the necessary details, i.e., the interface.
9 What is inheritance? Inheritance is the process of acquiring the properties of the exiting class into the new class. The existing class is called as base/parent class and the inherited class is called as derived/child class.
10 Explain the purpose of the keyword volatile. Declaring a variable volatile(易挥发的, 易变的) directs the compiler that the variable can be changed externally. Hence avoiding compiler optimization on the variable reference.
11 What is an inline function? A function prefixed with the keyword inline before the function definition is called as inline function. The inline functions are faster in execution when compared to normal functions as the compiler treats inline functions as macros.
12 What is a storage class? Storage class specifies the life or scope of symbols such as variable or functions.
13 Mention the storage classes names in C++. The following are storage classes supported in C++ : auto, static, extern, register and mutable
14 What is the role of mutable storage class specifier? The mutable specifier applies only to class objects. It allows a member of an object to override const member function. That is, a mutable member can be modified by a const member function.
15 Difference between Shallow and Deep copy of a class Shallow Copy stores the references of objects to the original memory address. Deep copy truly clones the underlying data. It is not shared between the first and therefore the copy. Deep copy is achieved using copy constructor and or overloading assignment operator.
16 What is a pure virtual function? A virtual function with no function body and assigned with a value zero is called as pure virtual function. E.g., virtual int area() = 0; from the class Shape
17 What is an abstract class in C++? A class with at least one pure virtual function is called as abstract class. We cannot instantiate an abstract class.
18 What is a reference variable in C++? A reference variable is an alias name for the existing variable. That means both the variable name and reference variable point to the same memory location. Therefore updating on the original variable can be achieved using reference variable too.
19 What is the role of static keyword on class member variable? A static variable does exit though the objects for the respective class are not created. Static member variable share a common memory across all the objects created for the respective class. A static member variable can be referred using the class name itself.
20 Explain the static member function. A static member function can be invoked using the class name as it does exist before class objects comes into existence. It can access only static members of the class.
21 Name the data type which can be used to store wide characters in C++. wchar_t
22 What are/is the operator/operators used to access the class members? Dot (.) and Arrow ( -> ). The dot operator is applied to the actual object. The arrow (->) operator is used with a pointer to an object.
23 Can we initialize a class/structure member variable as soon as the same is defined? No, Defining a class/structure is just a type definition and will not allocated memory for the same.
24 What is the data type to store the Boolean value? bool, is the new primitive data type introduced in C++ language.
25 What is function overloading? Defining several functions with the same name with unique list of parameters is called as function overloading. You cannot overload function declarations that differ only by return type.
26 What is operator overloading? Defining a new job for the existing operator w.r.t the class objects is called as operator overloading.
27 Do we have a String primitive data type in C++? No, it’s a class from STL (Standard template library).
28 Name the default standard streams in C++. cin, cout, cerr and clog.
29 Which access specifier/s can help to achieve data hiding in C++? Private & Protected.
30 When a class member is defined outside the class, which operator can be used to associate the function definition to a particular class? Scope resolution operator (::)
31 What is a destructor? Can it be overloaded? A destructor is the member function of the class which is having the same name as the class name and prefixed with tilde (~) symbol. It gets executed automatically w.r.t the object as soon as the object loses its scope. It cannot be overloaded and the only form is without the parameters.
32 What is a constructor? A constructor is the member function of the class which is having the same name as the class name and gets executed automatically as soon as the object for the respective class is created.
33 What is a default constructor? Can we provide one for our class? Every class does have a constructor provided by the compiler if the programmer doesn’t provides one and known as default constructor. A programmer provided constructor with no parameters is also called as default constructor. In such case compiler doesn’t provides the constructor.
34 Which operator can be used in C++ to allocate dynamic memory? ‘new’ is the operator can be used for the same.
35 What is the purpose of ‘delete’ operator? ‘delete’ operator is used to release the dynamic memory which was created using ‘new’ operator.
36 Can I use malloc() function of C language to allocate dynamic memory in C++? Yes, as C is the subset of C++
37 Can I use ‘delete’ operator to release the memory which was allocated using malloc() function of C language? No, we need to use free() of C language for the same.
38 What is a friend function? A function which is not a member of the class but still can access all the members of the class is called so. To make it happen we need to declare within the required class following the keyword ‘friend’.
39 What is a copy constructor? A copy constructor is the constructor which take the same class object reference as the parameter. It gets automatically invoked as soon as the object is initialized with another object of the same class at the time of its creation.
40 Does C++ supports exception handling? If so what are the keywords involved in achieving the same. C++ does supports exception handling. try, catch and throw are the keywords used for the same.
41 Explain the pointer – this. this, is the pointer variable of the compiler which always holds the current active object’s address.
42 What is the difference between the keywords struct and class in C++? By default the members of struct are public and by default the members of the class are private.
43 Can we implement all the concepts of OOPs using the keyword struct? Yes. struct class.
44 OOPs Concepts in C++ In Object Oriented Programming we write programs using classes and objects utilizing features of OOPs such as abstraction, encapsulation, inheritance and polymorphism.
45 What is the block scope variable in C++? A variable whose scope is applicable only within a block is said so. Also a variable in C++ can be declared anywhere within the block.
46 What is the role of the file opening mode ios::trunk? If the file already exists, its content will be truncated before opening the file.
47 What is the scope resolution operator (::)? The scope resolution operator is used to 1) Resolve the scope of global variables. 2) To associate function definition to a class if the function is defined outside the class.
48 What is a namespace? A namespace is the logical division of the code which can be used to resolve the name conflict of the identifiers by placing them under different name space.
49 What are command line arguments? The arguments which we pass to the main() function while executing the program are called as command line arguments. The parameters are always strings held in the second argument (below in args) of the function which is an array of character pointers. First argument represents the count of arguments (below in count) and updated automatically by operating system. main( int count, char *args[]) {}
50 What is a class template? A template class is a generic class. The keyword template can be used to define a class template.
51 How can we catch all kind of exceptions in a single catch block? The catch block with ellipses as follows catch(…) {}
52 What is keyword auto for? By default every local variable of the function is automatic (auto). In the below function both the variables ‘i’ and ‘j’ are automatic variables. void f(){ int i; auto int j;} NOTE − A global variable can’t be an automatic variable.
53 What is a static variable? A static local variables retains its value between the function call and the default value is 0. If a global variable is static then its visibility is limited to the same source code.
54 What is the purpose of extern storage specifier. Used to resolve the scope of global symbol main() { extern int i; cout<<i<<endl; }
55 What is the meaning of base address of the array? The starting address of the array is called as the base address of the array.
56 When should we use the register storage specifier? If a variable is used most frequently then it should be declared using register storage specifier, then possibly the compiler gives CPU register for its storage to speed up the look up of the variable.
57 Can a program be compiled without main() function? Yes, it can be compiled but cannot be executed, as the execution requires main() function definition.
58 Where an automatic variable is stored? Every local variable by default being an auto variable is stored in stack memory.
59 What is a container class? A class containing at least one member variable of another class type in it is called so.
60 What is a token? A C++ program consists of various tokens and a token is either a keyword, an identifier, a constant, a string literal, or a symbol.
61 What is a preprocessor? Preprocessor is a directive to the compiler to perform certain things before the actual compilation process begins.
62 What are the different ways of passing parameters to the functions? Which to use when? 1) Call by value − We send only values to the function as parameters. We choose this if we do not want the actual parameters to be modified with formal parameters but just used. 2) Call by address − We send address of the actual parameters instead of values. We choose this if we do want the actual parameters to be modified with formal parameters. 3) Call by reference − The actual parameters are received with the C++ new reference variables as formal parameters. We choose this if we do want the actual parameters to be modified with formal parameters.
63 What is reminder for 5.0 % 2? Error, It is invalid that either of the operands for the modulus operator (%) is a real number.
64 Which compiler switch to be used for compiling the programs using math library with g++ compiler? Opiton –lm to be used as > g++ –lm <file.cpp>
65 Can we resize the allocated memory which was allocated using ‘new’ operator? No, there is no such provision available.
66 Who designed C++ programming language? Bjarne Stroustrup.
67 Which operator can be used to determine the size of a data type/class or variable/object? sizeof()
68 How can we refer to the global variable if the local and the global variable names are same? We can apply scope resolution operator (::) to the for the scope of global variable.
69 What are valid operations on pointers? The only two permitted operations on pointers are i) Comparison ii) Addition/Subtraction (excluding void pointers)
70 What is recursion? Function calling itself is called as recursion.
71 What is the first string in the argument vector w.r.t command line arguments? Program name.
72 What is the maximum length of an identifier? Ideally it is 32 characters and also implementation dependent.
73 What is the default function call method? By default the functions are called by value.
74 What are available mode of inheritance to inherit one class from another? Public, private and protected
75 What is the difference between delete and delete[]? Delete[] is used to release the array allocated memory which was allocated using new[] and delete is used to release one chunk of memory which was allocated using new.
76 Does an abstract class in C++ need to hold all pure virtual functions? Not necessarily, a class having at least one pure virtual function is abstract class too.
77 Is it legal to assign a base class object to a derived class pointer? No, it will be error as the compiler fails to do conversion.
78 What happens if an exception is thrown outside a try block? The program shall quit abruptly.
79 Are the exceptions and error same? No, exceptions can be handled whereas program cannot resolve errors.
80 What is function overriding? Defining the functions within the base and derived class with the same signature and name where the base class’s function is virtual.
81 Which function is used to move the stream pointer for the purpose of reading data from stream? seekg()
82 Which function is used to move the stream pointer for the purpose of writing data from stream? seekp()
83 Are class functions taken into consideration as part of the object size? No, only the class member variables determines the size of the respective class object.
84 Can we create and empty class? If so what would be the size of such object. We can create an empty class and the object size will be 1.
85 What is ‘std’? Default namespace defined by C++.
86 What is the full form of STL? Standard template library
87 What is ‘cout’? cout is the object of ostream class. The stream ‘cout’ is by default connected to console output device.
88 What is ‘cin’? cin is the object of istream class. The stream ‘cin’ is by default connected to console input device.
89 What is the use of the keyword ‘using’? It is used to specify the namespace being used in. E.g., using namespace std;
90 If a pointer declared for a class, which operator can be used to access its class members? Arrow (->) operator can be used for the same
91 What is difference between including the header file with-in angular braces < > and double quotes "" If a header file is included with in < > then the compiler searches for the particular header file only with in the built-in include path. If a header file is included with in "", then the compiler searches for the particular header file first in the current working directory, if not found then in the built-in include path.
92 S++ or S = S+1, which can be recommended to increment the value by 1 and why? S++, as it is a single machine instruction (INC) internally.
93 What is the difference between actual and formal parameters? The parameters sent to the function at calling end are called as actual parameters while at the receiving of the function definition called as formal parameters.
94 What is the difference between variable declaration and variable definition? Declaration associates type to the variable whereas definition gives the value to the variable.
95 Which keyword is used to perform unconditional branching? goto
96 Is 068 a valid octal number? No, it contains invalid octal digits. (valid: 0, …, 7), no 8, just like binary numbers only have 0 and 1
97 What is the purpose of #undef preprocessor? It will be used to undefine an existing macro definition.
98 Can we nest multi line comments in a C++ code? No, we cannot.
99 What is a virtual destructor? A virtual destructor ensures that the objects resources are released in the reverse order of the object being constructed w.r.t inherited object.
100 What is the order of objects destroyed in the memory? The objects are destroyed in the reverse order of their creation.
101 What is a friend class? A class members can gain accessibility over other class member by placing the class declaration prefixed with the keyword ‘friend’ in the destination class.
