4. Conclusion & Next Work

Further work have to be done in order to gain much higher JPEG compression ratio. First, the epitome data (i.e., some row and column indices) which is necessary for the reconstruction should be compressed. Second, the error images between original ones and the reconstructed ones, can be saved as some other format files, or some threshold could be set before its JPEG compression.
