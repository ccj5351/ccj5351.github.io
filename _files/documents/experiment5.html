2. Result Analysis

Note:

  • The Epitome-related data only includes the Mean, without the row-column indices, since the indices can be learned during the inference process with the input image(s).
  • The residual images have been reduced, after an appropriate normalization is adopted, i.e. intensity 0 maps to 127, and the maximal absolute intensity (i.e., positive and nagative) maps to 255 and 0. So the residual iamges are encoded in 8-bit, i.e., 0 - 255 intensities.
  • Other encoding method with less bits will be tried to represent the residual images, in order to further reduce their size.
